# منصة الإتقان للقراءة - GitHub Ready

هذا المجلد جاهز للرفع مباشرة إلى GitHub ليعمل كمنصة ويب عبر GitHub Pages.

## الملفات:

- `student_firebase.html` : صفحة الطالب
- `teacher_firebase.html` : صفحة المعلم
- `shared_firebase.js` : الربط مع Firebase
- `firebase-config.js` : ضع إعدادات Firebase الخاصة بك هنا
- الشعار

## طريقة الرفع إلى GitHub Pages:
1. أنشئ مستودع جديد في GitHub.
2. ارفعي جميع الملفات الموجودة داخل هذا المجلد.
3. من إعدادات المستودع → Pages → اختاري الفرع main والمجلد root.
4. سيظهر رابط المنصة بعد ثوانٍ.
